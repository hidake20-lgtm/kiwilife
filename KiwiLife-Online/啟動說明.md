# Kiwi Life Online — 多人版啟動說明

## 第一次準備（只需做一次）
1. 安裝 Node.js LTS 版：https://nodejs.org

## 啟動伺服器（兩種方式擇一）
- **方式 A（最簡單）**：雙擊 `start.command`
  （若 macOS 阻擋：對檔案按右鍵 → 打開）
- **方式 B（終端機）**：
  ```
  cd 這個資料夾
  npm install   （只有第一次需要）
  npm start
  ```

## 開始遊玩
- 你自己：瀏覽器打開 **http://localhost:2567**
- 同 WiFi 的朋友：打開 **http://你的IP:2567**
  （查 IP：終端機輸入 `ipconfig getifaddr en0`）
- 每人輸入暱稱進入，就會在小鎮裡看到彼此！

## 遊戲內多人功能
- 👥 右上 HUD 顯示線上人數
- 看得到其他玩家走動＋頭上暱稱（6 種隨機外觀）
- 聊天：電腦按 **Enter**，手機按 **💬**，訊息變成頭上泡泡
- 伺服器沒開時會自動退回單人模式，照樣能玩

## 讓全世界的朋友連線（之後）
部署到 Render / Railway / Colyseus Cloud 即可，
伺服器已支援 `PORT` 環境變數，上傳整個資料夾就能跑。
